# Screenshots

This directory contains screenshots for the README.

## Required Screenshots

Add the following screenshots to this directory:

- `dashboard.png` - 机队状态仪表板
- `aircraft-detail.png` - 飞机详情页面
- `work-orders.png` - 工单管理页面
- `inventory.png` - 库存管理页面
- `maintenance-schedule.png` - 维保计划页面

## Recommended Dimensions

- Main screenshot: 1200 x 800 pixels
- Detail screenshots: 800 x 600 pixels

## How to Capture

1. Start the development server: `pnpm dev`
2. Navigate to the page
3. Use browser DevTools to set viewport size
4. Capture using OS screenshot tool or browser extension
