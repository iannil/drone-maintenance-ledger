## 1. 用户角色定义

### 1.1 角色层级

```
ADMIN (管理员)
    ├── MANAGER (机队经理)
    │   ├── INSPECTOR (检验员)
    │   └── MECHANIC (维修工)
    │       └── PILOT (飞手)  # 注：飞手独立于维修工，但在汇报关系中可能归属管理
```

### 1.2 角色职责概述

| 角色 | 全称 | 核心职责 | 主要工作场景 |
|------|------|----------|--------------|
| PILOT | 飞手 | 飞行作业、故障报告 | 飞行前检查、飞行记录、异常上报 |
| MECHANIC | 维修工 | 执行维修、更换零件 | 工单执行、零件领退、维修记录 |
| INSPECTOR | 检验员 | 质量把关、放行签字 | 工单检验、RII必检项、放行授权 |
| MANAGER | 机队经理 | 计划管理、资源协调 | 维保计划、工单分配、报表查看 |
| ADMIN | 管理员 | 系统管理、用户管理 | 组织配置、权限分配、系统设置 |

## 2. 飞手 (PILOT) 生命周期

```
┌─────────────────────────────────────────────────────────────┐
│                     飞手账户创建流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
```

### 2.2 日常作业流程

```
┌─────────────────────────────────────────────────────────────┐
│                     飞手日常作业流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐ │
│  │  飞行前检查    │ -> │  执行飞行任务  │ -> │  飞行后记录    │ │
│  └──────────────┘    └──────────────┘    └──────────────┘ │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  飞行前检查 (Pre-flight Check)                       │   │
│  │  ├── 查看飞机状态：是否可飞、是否在保、有无挂起故障        │   │
│  │  ├── 查看零部件状态：关键件寿命、到期预警                │   │
│  │  ├── 检查证件有效期：适航证、电台执照、保险               │   │
│  │  ├── 查看天气/空域限制（如有集成）                       │   │
│  │  └── 数字化检查单：逐项打钩确认                          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  执行飞行任务                                          │   │
│  │  ├── 起飞                                             │   │
│  │  ├── 飞行作业（巡检/物流/航拍等）                       │   │
│  │  ├── 降落                                             │   │
│  │  └── (可选) 接入飞控日志自动解析                       │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  飞行后记录                                            │   │
│  │  ├── 填写飞行日志：日期、起降地点、时长、起降次数         │   │
│  │  ├── 记录任务性质：训练/作业/试飞                       │   │
│  │  ├── 上传飞行数据/照片（如有）                          │   │
│  │  └── 系统自动更新飞机/零部件使用数据                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**相关功能点**:
- 飞机状态查询 (`GET /aircraft/:id/status`)
- 零部件查询 (`GET /components?aircraftId=:id`)
- 维保到期查询 (`GET /maintenance/schedule/due`)
- 飞行记录创建 (`POST /flight-logs`)
- 检查单模板 (`GET /checklists/pre-flight`)

### 2.3 故障报告流程 (PIREP)

```
┌─────────────────────────────────────────────────────────────┐
│                    故障报告流程 (PIREP)                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [飞手] 发现异常                                            │
│       │                                                     │
│       ├── 现象描述：声音异常、震动、告警等                      │
│       ├── 故障等级判断：                                      │
│       │   ├── AOG (Aircraft On Ground) - 停飞               │
│       │   ├── 监视使用 - 可飞但需关注                          │
│       │   └── 稍后处理 - 非关键故障                           │
│       ├── 上传证据：照片、录音、视频                           │
│       └── 提交报告                                           │
│                                                             │
│  [系统] 自动处理                                            │
│       │                                                     │
│       ├── 飞机状态自动更新为 GROUNDED (如标记AOG)              │
│       ├── 创建/关联维修工单                                   │
│       ├── 通知相关维修工和检验员                              │
│       └── 记录故障报告到履历                                  │
│                                                             │
│  [飞手] 后续跟进                                            │
│       │                                                     │
│       ├── 查看工单进度                                        │
│       ├── 配合维修说明故障详情                                │
│       └── 确认放行后恢复飞行                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**相关功能点**:
- 故障报告创建 (`POST /pilot-reports`)
- 工单自动创建 (`POST /work-orders`)
- 飞机状态更新 (`PUT /aircraft/:id/status`)
- 通知服务 (Notification Service)
- 工单查询 (`GET /work-orders`)
